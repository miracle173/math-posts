# Notation

## Locations
    box         B(i,j)      i = 0..2, j = 0..2
    cell        b(i,j;k,n)  i = 0..2, j = 0..2, k = 0..2, l = 0..2
                b(i,j,k,n)  i = 0..2, j = 0..2, k = 0..2, l = 0..2
                b(i; j)     i = 0..8, j = 0..8
                b(i, j)     i = 0..8, j = 0..8
                b(i,j;k)    i = 0..2, j = 0..2, k = 0..8
                b(i;j,k)    i = 0..8, j = 0..2, k = 0..2
    box row     R(i)        i = 0..2
    cell row    r(i,j)      i = 0..2,j = 0..2
                r(i)        i = 0..8
    box col     C(i)        i = 0..2
    cell col    c(i,j)      i = 0..2, j = 0..2
    col         c(i)      i = 0..8

a box row iscalled band, a box column is called stack. The values in the cells of a sudolu are called clues.
 ## Transformations
 
### Rotation
    r0 = id
    r1   rotation by pi/2, counterclockwise
    r2 = r1^2
    r3 = r1^3
    

### Reflection
    d1  /
    d2  |
    d3  \
    d4  -
    
### box row Permution
    pR(s)   s permutation
    
### box col permution
    pC(s)   s permutation
    
### cell row permutions
    pr(i, s)   i = 0..2 (box row index), s permutation of 0,1,2
    pr(i,j)
    
### cell col permutions
    pc(i, s)   i = 0..2 (cell row index), s permutation of 0,1,2
